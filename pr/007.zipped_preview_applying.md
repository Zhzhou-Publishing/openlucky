# 压缩预览功能增加功能2

## 不拷贝 .preset.json

1. PhotoGallery.vue 不向 originalDirectory 拷贝 .preset.json。
2. PhotoEdit.vue 不向 originalDirectory 拷贝 .preset.json。 

## 函数1：单个应用到实际文件，新增指定 .preset.json 读取配置并应用文件的函数

1. 参数1，preset_file，指定一个 .preset.json 文件的路径。
2. 参数2，input_file_path，指定一个输入文件路径。
3. 参数3，output_file_path，指定一个输出文件路径。
4. 函数执行大致过程：
   1. 读取 preset_file 的内容，保存一个 object 到 preset_obj。
   2. 按照 input_file_path 的文件名，查找 preset_obj 里面对应文件名为 key 的 value。注意查找的时候如果是 RAW 格式的图片，则应该在文件末尾再加一个 .tif 后缀（最好是这三种都查一遍以保险一些：RAW 格式原文件，原文件名 + .tif 后缀，原文件名 + .tiff 后缀）。preset_obj 的内容请参考 presets.sample.json 文件。
   3. 执行 openlucky filmparam --input [input_file_path] --output [output_file_path] --preset [preset_params] 这个命令。
   4. preset_params 的构成是这个格式："mask_r,mask_g,mask_b,gamma,contrast"，各类参数来自于你读到的 preset_obj 的一个值。

## 函数2：批量应用到实际文件，新增指定 .preset.json 读取配置并批量应用文件的函数

1. 参数1，preset_file，指定一个 .preset.json 文件的路径。
2. 参数2，input_dir，指定一个输入文件夹。
3. 参数3，output_dir，指定一个输出文件夹。
4. 函数执行大致过程：
   1. 读取 preset_file 的内容，保存一个 object 到 preset_obj。
   2. 遍历 input_dir 的所有图片文件（包括 RAW 和非 RAW 格式的所有图片）。
   3. 按照遍历到的图片文件名，查找 preset_obj 里面对应文件名为 key 的 value。注意查找的时候如果是 RAW 格式的图片，则应该在文件末尾再加一个 .tif 后缀（最好是这三种都查一遍以保险一些：RAW 格式原文件，原文件名 + .tif 后缀，原文件名 + .tiff 后缀）。preset_obj 的内容请参考 presets.sample.json 文件。
   4. 遍历的时候拿到值过后，挨个执行 openlucky filmparam --input [input_dir/looped_filename] --output [output_dir/looped_filename] --preset [preset_params] 这个命令。
   5. preset_params 的构成是这个格式："mask_r,mask_g,mask_b,gamma,contrast"，各类参数来自于你读到的 preset_obj 的一个值。

## PhotoEdit.vue 页面改造

1. 隐藏 apply 按钮，但是保留它和 Apply 操作函数，我将评估一下以后的版本是否移除。
2. 新增 Save All 按钮，放在 Apply All 按钮右边，颜色为天蓝色。
3. Save All 按钮的操作函数是 saveAll，saveAll 函数执行上面的函数2的过程。
4. 在 SaveAll 的开始到结束过程中，所有图片都设置为 affected，禁止变更 input1-5，禁止点击任何按钮，禁止切换图片。
5. saveAll 函数执行函数2时，传入参数如下：
   1. preset_file：workingDirectory/.preset_file.json
   2. input_dir：originalDirectory
   3. output_dir：originalDirectory/output

## PhotoGallery 页面改造

1. 新增 Save All 按钮，放在 Apply 按钮右边，颜色为天蓝色。
2. Save All 按钮的操作函数是 saveAll，saveAll 函数执行上面的函数2的过程。
3. 在 SaveAll 的过程中，设置 preset 下拉框、Apply 按钮和 Save All 按钮为禁止状态，同样设置所有 Image 的鼠标为 waiting 状态。
4. PhotoGallery 中的 Save All 按钮放在 BottomMenuBar 这个 component 里实现，禁止使用 slot 的方式。

## 关于两个页面的 Save All 按钮

1. 将这个按钮提出来作为 component，后续需求会修改它的统一样式。
2. 两个页面的 Save All 按钮，都设置一个 Ctrl + S 的快捷键，并在 hover 的时候给出提示。
3. Save All 有一个全局状态 isSaveAllClicked，初始时候这个状态为 false，当点击两个页面内任意一个的 Save All 过后状态将设置为 true。
4. 再次将 isSaveAllClicked 再次设置为 false 的时机是，当如下任一事件发生时：
   1. PhotoGallery 执行过 applyPreset；
   2. PhotoEdit 页面执行过 apply, applyPreview, applyAll 其中之一。
