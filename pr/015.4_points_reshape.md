# 四点形状校正

四点校正的目的是从胶片大图裁切出有效的图片。用户在图片中点击

## 命令行工具

### 命令

openlucky tool reshape

### 参数

| 参数     | 短参数 | 有效值                            | 说明           |
| -------- | ------ | --------------------------------- | -------------- |
| --input  | -i     | 输入文件地址                      | 输入文件地址   |
| --output | -o     | 输出文件地址                      | 输出文件地址   |
| --point1 | -p1    | 横纵坐标，如 10,13                | 左上角裁切锚点 |
| --point2 | -p2    | 横纵坐标，如 100,14               | 右上角裁切锚点 |
| --point3 | -p3    | 横纵坐标，如 101,134              | 右下角裁切锚点 |
| --point4 | -p4    | 横纵坐标，如 8,132                | 左下角裁切锚点 |
| --shape  | -s     | 画布宽int,画布高int，如 6000,4000 | 输出的宽高形状 |

### 执行过程

1. p1->p2->p3->p4->p1 的路径上框出一个不规则的四边形，需检查描绘是否是凸四边形。
2. 按照这个路径抠出有效的图片。
3. 按照 -s 的形状创建白色空画布。
4. 将 p1 p2 p3 p4 点拉伸到画布的左上 右上 右下 左下，覆盖整个画布。
5. 输出。

### 样例

openlucky tool reshape -i somepic.tif -o somepic_out.tif --point1 10,13 --point2 100,14 --point3 101,133 --point4 8,132 -s 6000,4000

### 注意事项

1. 输入参数需校验是否是凸四边形。
2. 需同时支持 8bit 常见位图（jpeg png 等），16bit tiff 图，和常见的 RAW 格式图像。详细格式参考 image_formats.py
