# 压缩预览功能增加功能3

## 全局状态 Save All 红点

1. 全局状态 isSaveAllClicked 如果是 false 的时候所有的 Save All 按钮都加一个 Apply 按钮同款的红点。
2. 全局状态 isSaveAllClicked 如果是 true 的时候让这个红点消失。

## Save All 的 Disable 状态补充

1. PhotoGallery 页面里 Save All 按钮和 Apply 按钮的 Disable 与否受到 saveAll 和 applyPreset 两个函数的控制。
2. PhotoEdit 页面里 Save All 按钮、已隐藏的 Apply 按钮和 Apply All 按钮的 Disable 与否受到 apply，applyPreview，applyAll 三个函数的控制。
3. 上面两个页面的多个按钮的 Disable 状态，请尽可能复用一些状态，删除一些不必要的冗余状态。
