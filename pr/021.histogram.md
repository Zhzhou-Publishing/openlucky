# PhotoEdit 界面上做直方图采样时机

## 问题

还是在采样底片的直方图，还是没有实现【只在 apply 过的时候采样转正过的图片】。

## 我设想的方案

进入 PhotoEdit 页面如果发现当前图片是 apply 过的就执行一次直方图采样，否则就不进行采样。每次 apply 完成过后再进行直方图采样。

## 方案详细过程可参考

1. 直方图有两个状态：isHistogramLoading 和 isHistogramDisplay。

2. 进入页面或者切换 thumbnail 的初始状态：isHistogramLoading=false isHistogramDisplay=false。此时如果发现图片是被 apply 过的，检查当前图片是否缓存了histogram的数据，如果没有缓存则开始采样当前的图片的直方图数据，如果有缓存则直接使用缓存、渲染直方图。

3. 进入页面或者切换 thumbnail 此时如果发现当前图片没有 apply 过或者被重置过：isHistogramLoading=true isHistogramDisplay = false

4. 当 apply/applyAll 开始执行的时候：isHistogramDisplay=true isHistogramLoading=true

5. 当 apply/applyAll 成功的时候：开始执行直方图采样的命令。

6. 当直方图采样出结果的时候：渲染直方图、更新直方图缓存、isHistogramDisplay=true isHistogramLoading=true
