# PhotoEdit.vue 新样式

## thumbnail 显示位置变更

1. thunmbnail 显示区域放在屏幕左侧、竖排，header 区域放在新 thumbnail 区域的上方，input1-5和操作按钮区域放在 thumbnail 区域的右边，主图区域放在 thumbnail 区域的右边。
2. 取消左右方括号、左右方向键换张的快捷键。
3. 新增上下方向键换张的快捷键。

设计草图如下：
┌─────────────────────────────────────┐
│            Header                   │
├───────┬─────────────────────────────┤
│       │                             │
│Thumbs │    Main Pic                 │
│       │                             │
│       │                             │
│       ├─────────────────────────────┤
│       │   Option Area               │
└───────┴─────────────────────────────┘

## Header 栏和操作栏固定、主图区大小可变

1. 本页 header 栏目固定到顶部的位置。
2. Option Area 栏固定到底部的位置。
3. 左侧的图片 thumbnail 区的高度一直从 header 栏目之下、到页面最底部。
4. 页面中间主图随窗体大小可变。
5. 注意1：本页面是运行在 electron 窗体的。
6. 注意2：header 栏和 Option Area 栏不要遮盖到主图和 thumbnail 栏的一些图片，做好避让。
7. 注意3：特别检查你的变更，不要把 Option Area 栏挤到页面外去了。
8. 注意4：Main Pic 和 Option Area 区域各自单独是一个 box，Main Pic + Option Area 高度 = electron窗体页面内容部分总高度 - App.vue的NavBar高度 - 本页header高度。
9. 注意5：Main Pic 的 img 标签图片不要被 Option Area 遮挡，应该严格限制好高度。Main Pic 图片高度 = electron窗体页面内容部分总高度 - App.vue的NavBar高度 - 本页header高度 - Option Area 高度。
