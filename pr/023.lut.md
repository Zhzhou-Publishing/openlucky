# 快速查找表 LUT 实现快速运算

## 生成 lut 和查找 lut 优化标准

### 组织目录查找层级

1. 生成 lut 的时候将点号切开作为层级目录放入目标目录
   
   比如 com.github.zhzhou-publishing.openlucky.s-curve.power-curve.p-0_30-k-0_50.lut

   放在 lut>com>github>zhzhou-publishing>openlucky>s-curve>power-curve 下，文件名为 p-0_30-k-0_50.lut

2. 查找 lut 的时候按照上述的点号层级在目录里查找

### 带参数的文件名支持负号

负号前缀是 `n`

如 com.github.zhzhou-publishing.openlucky.common.apply-exposure-7ev.ev-n7_00.lut 表示传入参数为 ev=-7.0

## process_film 封装曝光函数的 LUT

最大兼容到 16bit 的色深，可向下兼容到 8bit。

1. apply_exposure_3ev

   注册函数名：common.apply-exposure-3ev

   封装 [-3, 3] 步进为 0.1 的 LUT

2. apply_exposure_5ev 

   注册函数名：common.apply-exposure-5ev

   封装 [-5, 5] 步进为 0.1 的 LUT

3. apply_exposure_7ev

   注册函数名：common.apply-exposure-7ev

   封装 [-7, 7] 步进为 0.1 的 LUT

## process_film 的 gamma 运算封装成 LUT

最大兼容到 16bit 的色深，可向下兼容到 8bit。

注册函数名：common.gamma

封装 [0.4, 3] 步进为 0.1 的 LUT

函数过程主要是这个：img = np.power(img, preset_gamma)
