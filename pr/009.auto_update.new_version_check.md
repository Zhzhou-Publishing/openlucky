# 自动更新功能 - 新版本检查

## 新版本检查

1. 通过 Github API 检查新版本：https://api.github.com/repos/Zhzhou-Publishing/openlucky/releases/latest
2. 网页里面的版本号字段是：name
3. 每次启动程序的时候通过后台程序检查一次 Github API，对照 name 与当前版本字段（当前版本号可通过 electron 的 getVersion 获取），如果不一致，则弹窗告知用户有新版本，版本号是什么。
4. 检查频次为每小时区间内一次，如果当前小时区间访问过这个 API，那么下一次检查就在下一小时区间内。比如当前时间是 17:33:25，那么从此刻开始到 17:59:59 前，启动的时候都不再做检查。
5. 弹窗的内容如下：
   1. 新版本号（name 字段）。
   2. 发布时间（published_at 字段）。
   3. 两个按钮：立即下载、暂不下载（点击立即下载后用系统默认浏览器访问 html_url 的网页）。
6. 如果 Github 无法访问则弹出警告：无法检查更新，请确认网络环境。
7. 注意弹窗做好 i18n。
