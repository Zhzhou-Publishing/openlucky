# OpenLucky 桌面应用 - 更新日志 (feature/desktop-app)

## 版本 v1.2.0-rc1

此版本带来了完整的桌面应用程序体验，具备胶片处理能力、预设管理和精致的界面。

---

## 新功能

### 桌面应用程序
- **Electron 桌面应用**：基于 Vue.js 的完整桌面应用，提供无缝的用户体验
- **自定义应用图标**：OpenLucky 品牌标识，包含自定义图标资源（.ico 和 .png 格式）
- **窗口管理**：固定窗口大小，自动隐藏菜单栏，可调整大小的切换和开发者工具控制
- **Windows 安装程序**：Inno Setup 配置，用于创建专业的 Windows 安装程序，包含桌面和开始菜单快捷方式
- **构建系统**：完整的构建基础设施，支持便携版和安装版

### 照片画廊和导航
- **照片画廊页面**：从选定目录浏览图片，采用缩略图网格布局
- **目录选择**：内置目录选择器，显示文件列表
- **缩略图生成**：使用 sharp 库自动生成缩略图，支持多种图片格式，包括 TIF/TIFF
- **图片模态框**：全屏图片查看，右键触发
- **键盘导航**：使用方向键和方括号键在图片之间导航

### 胶片处理
- **自定义参数处理**：新增 `filmparam` 和 `filmparambatch` 命令，支持自定义参数处理
  - 支持自定义 mask_r、mask_g、mask_b、gamma 和 contrast 值
  - 支持整个目录的批处理能力
- **预设管理**：
  - 自动将预设保存到 `.preset.json`，带时间戳格式
  - 支持动态预设加载，使用 `.preset.json`
  - 预设应用支持实时更新
  - 分离工作目录和预览目录，优化工作流程
- **处理反馈**：预设应用期间的加载状态和动画

### PhotoEdit 页面
- **全分辨率图片查看器**：查看和编辑高分辨率图片，带参数控制
- **NumberInput 组件**：自定义输入组件，带有增减按钮和键盘快捷键
  - 5 个参数控制：Mask-R、Mask-G、Mask-B、Gamma、Contrast
  - 可配置的步进值，支持精细调整
  - 键盘快捷键工具提示，提升无障碍体验
- **应用按钮**：
  - Apply 按钮用于单张图片处理（Enter 键）
  - Apply All 按钮用于批量处理（Ctrl+Enter）
  - 处理操作期间的视觉反馈
  - 操作期间禁用状态管理
- **即时更新**：使用时间戳绕过缓存更新图片（无加载遮罩）
- **窗口调整大小**：进入/离开画廊视图时动态调整窗口大小

### 用户界面
- **底部菜单栏**：预设选择器，带应用按钮和视觉指示器
- **功能卡片**：主页带有不同功能的导航卡片
- **导航栏**：简洁的导航，包含主页和关于链接
- **响应式设计**：自适应布局和间距

### 键盘快捷键
- **全局快捷键**：
  - 方向键：在图片之间导航
  - [ / ]：在图片之间导航（替代方式）
  - Enter：应用当前设置
  - Ctrl+Enter：应用到所有图片
- **参数控制**：
  - Q/A：增加/减少 Mask-R
  - W/S：增加/减少 Mask-G
  - E/D：增加/减少 Mask-B
  - R/F：增加/减少 Gamma（小步进）
  - T/G：增加/减少 Contrast（小步进）
  - Alt+Shift+R/F：增加/减少 Gamma（大步进）
  - Alt+Shift+T/G：增加/减少 Contrast（大步进）

### 配置和安装
- **Config 命令**：读取应用程序配置
- **安装检查**：在启用功能之前验证 openlucky CLI 是否可用
- **环境文档**：BUILD.md 中的完整构建说明
- **Python 支持**：PYTHONPATH 要求的文档

## 改进

### 性能
- **并行加载**：使用 Promise.all 并行加载图片和预设
- **缓存**：使用图片 URL 时间戳绕过 Electron 内置浏览器缓存
- **加载状态**：异步操作期间的视觉指示器

### 用户体验
- **视觉反馈**：未应用预设的红点指示器
- **动画**：底部菜单的滑入动画
- **禁用状态**：操作期间正确禁用控件
- **错误处理**：优雅的错误消息和后备状态
- **无障碍性**：键盘快捷键在所有输入和控件上工作

### 代码质量
- **组件结构**：可重用组件（NumberInput、FeatureCard、Navbar）
- **状态管理**：使用 Vue 3 Composition API 的干净响应式状态
- **IPC 通信**：主进程和渲染进程之间可靠的进程间通信
- **类型安全**：一致的 prop 类型和验证

## Bug 修复

- **滚动支持**：修复缩略图画廊中的滚动问题
- **图标一致性**：更新图标文件名引用为 OpenLuckyApp.exe
- **模块导入**：为 Python 模块导入添加 lib. 前缀
- **路径处理**：在 JSON 路径中使用正斜杠以避免双反斜杠
- **DevTools 控制**：防止在生产版本中自动打开 DevTools
- **窗口阻塞**：隐藏预览窗口以防止执行阻塞
- **图片模态框触发**：改为仅右键触发以改善用户体验
- **浏览器默认输入**：从数字输入中移除默认的微调按钮
- **快捷键处理**：修复输入聚焦时键盘快捷键无法工作的问题
- **预设应用**：改进预设加载和应用逻辑

## 技术细节

### 新增依赖
- **electron**：桌面应用框架
- **electron-builder**：应用程序打包
- **vue-router**：客户端路由
- **sharp**：图片处理和缩略图生成

### 构建系统
- **npm scripts**：CLI 和桌面应用的构建命令
- **electron-builder config**：Windows 打包配置
- **Inno Setup**：专业的 Windows 安装程序创建
- **.gitignore**：构建工件的正确排除

### 文件结构
```
app/                          # Electron 应用程序
├── src/
│   ├── components/           # Vue 组件
│   │   ├── NumberInput.vue
│   │   ├── Navbar.vue
│   │   └── FeatureCard.vue
│   ├── pages/              # Vue 页面
│   │   ├── Home.vue (已移除)
│   │   ├── PhotoDirectory.vue
│   │   ├── PhotoGallery.vue
│   │   └── PhotoEdit.vue
│   ├── App.vue
│   └── main.js
├── package.json
└── (构建文件)

dist-electron/              # 构建的 Electron 应用
├── win-unpacked/
└── (安装程序文件)
```

### 配置文件
- **OpenLucky.iss**：Inno Setup 安装程序配置
- **app/package.json**：Electron 应用配置
- **.preset.json**：运行时预设存储（生成）
- **presets.sample.json**：预设格式参考
