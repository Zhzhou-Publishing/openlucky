# Changelog

## v1.4.1-rc1 (2026-04-09)

### Features
- **Preview Preparation Progress Display**: Shows real-time progress when preparing preview images
- **Image Navigation Optimization**: Use Ctrl + arrow keys to switch between images for more intuitive navigation
- **Performance Improvements**: Enhanced image processing speed for faster preview preparation

---

## Installation

For Windows users using the installer, simply run the `openlucky-setup.exe` file and follow the installation wizard.

For development or manual installation, please refer to the project documentation.

---

## Upgrade Notes

This release improves user experience by adding real-time progress display during preview preparation, so you can clearly see the current processing progress. Additionally, image navigation has been optimized with Ctrl + arrow keys for easier image switching. Performance improvements also make the entire preview preparation process smoother.
