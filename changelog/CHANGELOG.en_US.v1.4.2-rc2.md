# v1.4.2-rc2

## New Features

- **macOS Native Support**: Added native macOS application with Apple Silicon (arm64) support
- **Automatic Configuration Management**: Default configuration file is automatically created on first launch
- **Unified Build Process**: Optimized Windows and macOS application packaging workflow

## Improvements

- Improved application startup performance (macOS)
- Enhanced build workflow with cross-platform parallel builds
- Enhanced security: Upgraded to SHA256 checksums
- Optimized version number format for standard compliance
