# Changelog for v1.4.3-rc.4

## Dark/Light Theme

- Added dark and light theme support, synced with the system appearance.
- The window now maximizes automatically when opening.
- Fixed slider visibility issue in light mode.

## Histogram Improvements

- Improved histogram overlay with padding and dark background.
- Added caching for histogram results to avoid redundant sampling.
- Fixed histogram sampling triggering on un-applied images.
- Fixed mask color picking to use the correct working-directory file.
- Deferred histogram display until image is applied.
