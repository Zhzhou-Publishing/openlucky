# Changelog for v1.4.3-rc.5

## Photo Directory

- Added inline progress display and cancel support during directory loading.
- Added compress preview toggle for faster thumbnail generation.
- Fixed layout issues when resizing the tool window.

## Photo Edit

- Added zoom and pan to the main image using Ctrl+scroll, pinch gestures, and keyboard shortcuts.
- Added 180-degree rotation option in the right-click context menu.
- Restored manual Apply button — parameters no longer auto-apply on change.
- Fixed Apply completion interfering with other images when the user has switched thumbnails.
