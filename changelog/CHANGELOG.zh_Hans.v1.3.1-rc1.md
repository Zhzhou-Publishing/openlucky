# OpenLucky v1.3.1-rc1

## 新功能

### 自动更新检查功能
- 添加应用启动时的自动更新检查
- 实现版本检查器模块（`versionChecker.js`），查询 GitHub API 获取最新版本
- 发现新版本时显示更新通知对话框
- 集成 shell 模块，在默认浏览器中打开发布页面供用户下载
- 实现每小时检查一次，避免频繁的 API 调用
- 添加网络连接问题的错误处理
- 为更新通知和网络错误提供用户友好的对话框
- 在用户数据目录中存储上次检查时间戳以保持持久性

### 代码质量与维护
- 将所有配置文件中的版本更新至 v1.3.1-rc1
- 添加更新检查过程的全面日志记录
- 改进错误消息和用户反馈
