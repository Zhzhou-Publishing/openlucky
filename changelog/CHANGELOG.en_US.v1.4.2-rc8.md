# OpenLucky v1.4.2-rc8

## New Features

- **Four-point Perspective Correction**: Added a reshape tool for correcting perspective distortions in images using four-point adjustment
- **Per-channel Contrast Adjustment**: Added support for adjusting red, green, and blue channels separately for more precise color control
- **Batch Processing Enhancements**: Extended batch processing to support per-channel contrast adjustments
- **Improved UI Layout**: Enhanced photo edit interface with better spacing and dynamic height calculation
- **Tabbed Interface**: Organized photo edit parameters into basic and advanced tabs for better usability

## Improvements

- Enhanced TIFF file handling with improved compatibility
- Optimized processing stability with fixed concurrency settings
- Streamlined parameter organization in the photo editor

## Bug Fixes

- Resolved CPU core detection issues for more consistent performance
