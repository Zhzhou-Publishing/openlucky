# Changelog for v1.4.3-rc.7

## Tone Adjustment

- New Tone Adjustment tab lets you control the tonal curve — adjust pivot point and curve shape to fine-tune highlights, midtones, and shadows.
- Added `--tone auto` option to automatically derive the best tone settings from each image's histogram.

## Film Processing

- Added gamma alignment stage for more consistent color rendition across different input formats.
- S-curve LUT support with faster pre-computed lookup tables.

## Interface

- The histogram overlay now scales more consistently regardless of image content.
- Added a prompt when the system language differs from the app language, making it easier to switch.
